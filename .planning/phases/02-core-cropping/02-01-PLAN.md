  <files>app/composables/useCropWindow.ts</files>
  <action>
Create composables/useCropWindow.ts implementing:

2. isDragging, isResizing, activeHandle (nw/ne/sw/se) refs
3. startDrag(e: MouseEvent) - records start position, sets isDragging

6. onMouseUp - resets all state
7. Aspect ratio constraint (per RESEARCH Pattern 3):
   - NW handle: adjust x AND y to keep SE corner stationary
   - NE handle: adjust y to keep SW corner stationary
   - SW handle: adjust x to keep NE corner stationary
   - SE handle: only adjust width/height (opposite corner is already stationary)
8. clampToBounds: x in [0, imageWidth - width], y in [0, imageHeight - height]
9. Global mousemove/mouseup listeners attached in onMounted, removed in onUnmounted
10. SSR-safe with process.client guard

Per CROP-03: ratio constraint maintained during resize by adjusting BOTH dimensions AND position for NW/NE/SW handles.
Per CROP-06: all positions clamped to image bounds.
</action>
