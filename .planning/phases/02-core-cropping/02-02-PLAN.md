  <action>
Create components/CropOverlay.vue implementing:

1. Props: imageUrl, imageWidth, imageHeight, cropX, cropY, cropWidth, cropHeight, overlayOpacity
2. Layout:
   - Wrapper div: position relative, contains image and overlay
   - img element: displays the image at natural size (max-width: 100%)
   - Overlay div: position absolute, inset 0, contains:
     - Dark mask: uses clip-path polygon to show area OUTSIDE crop window
     - Crop window: white 2px border box (per CROP-05)
     - 4 CropHandle components at corners

3. Dark mask calculation using clip-path polygon:
   - Top-left to top-right, down to cropY, right to cropX, down to cropY, right to cropX+cropWidth, down to cropY+cropHeight, left to cropX, up to cropY, left to 0, up to 100%
   - Background: rgba(0,0,0,overlayOpacity) where opacity is 0.65 (per D-07)

4. Crop window: position absolute, left=cropX, top=cropY, width=cropWidth, height=cropHeight
5. White border: box-shadow: 0 0 0 2px white (per CROP-05)
6. Window drag: @mousedown on window calls useCropWindow().startDrag()
7. Handle drag: @mousedown.stop on each handle calls useCropWindow().startResize(handle)
8. pointer-events: none on mask, pointer-events: auto on window and handles

Per CROP-01: Draggable crop window overlay.
Per CROP-04: Area outside crop window dimmed.
Per CROP-05: White border on crop window.
Per D-07: Dark overlay at 60-70% opacity.
